# 第七部分：部署与基础设施

一人公司的运维哲学可以浓缩为三个零：零停机、零手工操作、零不必要支出。Clipboard Inspector 当前跑在 GitHub Pages 上，构建和部署完全自动化，每月成本为零。这已经是很好的起点，但随着用户增长和功能扩展，基础设施需要跟上节奏。

本部分从四个层面梳理部署策略。第 7.1 节对比主流托管平台的免费层，判断何时迁移、迁到哪里。第 7.2 节设计 CI/CD 流水线，把当前的分散工作流合并为更高效的单一管线。第 7.3 节规划监控和分析方案，用免费工具覆盖流量统计、错误追踪和可用性监控。第 7.4 节讨论域名、SSL 和性能优化，把基础设施成本控制在每年 $12 起步。

所有方案以"免费优先、按需升级"为原则，确保每个阶段的支出都有对应的收入支撑。
