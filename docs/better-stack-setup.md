# Better Stack 监控配置指南

## 概述

Better Stack 提供免费的网站可用性监控服务（免费层：10 个监控器，3 分钟检测间隔，支持自定义域名的状态页面）。本指南说明如何为 Clipboard Inspector 配置监控。

## 注册与配置步骤

### 1. 注册账号

访问 https://betterstack.com/ 使用 GitHub 账号或邮箱注册。免费层无需信用卡。

### 2. 创建监控器

1. 登录后点击 "Monitors" → "Create monitor"
2. 选择类型：**Heartbeat**（针对静态站点，检测页面可访问性）
3. 配置参数：

| 参数       | 值                                                    |
| ---------- | ----------------------------------------------------- |
| 名称       | Clipboard Inspector                                   |
| URL        | `https://fudesign2008.github.io/clipboard-inspector/` |
| 检测间隔   | 3 分钟（免费层最低）                                  |
| 超时       | 30 秒                                                 |
| 预期状态码 | 200                                                   |

### 3. 配置告警

免费层支持以下告警渠道：

-   Email（默认）
-   Slack（通过 Webhook）
-   短信（有限次数）

**推荐配置**：

-   主邮箱：你的 Gmail
-   连续 2 次失败后发送告警
-   恢复后发送恢复通知

### 4. 状态页面（可选）

Better Stack 免费层提供一个状态页面，可配置自定义域名：

-   访问 `https://your-username.betteruptime.com`
-   在 Settings 中绑定自定义域名（需要 DNS CNAME 记录）

### 5. 添加到策略书监控章节

配置完成后，将以下信息记录到运营文档中：

-   监控器 ID
-   状态页面 URL
-   告警邮箱

## 与其他监控工具对比

| 工具             | 免费监控数 | 检测间隔 | 状态页自定义域名 | 优势                   |
| ---------------- | ---------- | -------- | ---------------- | ---------------------- |
| **Better Stack** | 10         | 3 分钟   | 支持             | 界面现代，免费层最慷慨 |
| UptimeRobot      | 50         | 5 分钟   | 基础版           | 监控数量多             |
| Pingdom          | 1          | 1 分钟   | 不支持           | 检测频率高             |

## 成本

免费层完全够用。当需要超过 10 个监控器时，付费层为 $29/月。

Clipboard Inspector 目前只需要 1 个监控器（主站），因此 **$0/月**。
