# 隐私政策

生效日期：2026年4月25日

## 概述

Clipboard Inspector（以下简称"本工具"）是一款纯客户端的网页工具，用于检查和查看剪贴板内容。本隐私政策说明本工具如何处理用户数据。

## 数据处理方式

### 本地处理

本工具的所有数据处理均在用户浏览器本地完成。剪贴板内容的读取、解析、展示和导出操作均不会将任何数据传输到外部服务器。

### 不收集数据

本工具不收集、不存储、不传输任何用户数据，包括但不限于：

- 剪贴板内容
- 个人身份信息
- 使用日志
- 设备信息
- IP 地址

### 无 Cookie

本工具不使用 Cookie 或任何类似的追踪技术。

### 无第三方数据共享

我们不会将任何数据共享给第三方，因为我们本身不收集任何数据。

## 技术说明

本工具通过浏览器原生 API（DataTransfer / ClipboardItem）读取剪贴板内容。这些操作完全在浏览器沙箱环境中进行，数据不会离开用户的设备。

## 托管服务

本工具托管于 GitHub Pages。GitHub Pages 由 Microsoft 提供基础设施服务。关于 GitHub Pages 的数据处理实践，请参阅 [GitHub 隐私声明](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement)。

## 用户权利（GDPR）

尽管本工具不收集任何个人数据，我们仍尊重用户的隐私权利：

- **知情权**：本政策说明了我们如何处理数据（即：不处理）。
- **访问权**：由于不收集数据，不存在可供访问的个人数据。
- **删除权**：由于不存储数据，无需请求删除。
- **可携带权**：不适用，因为无数据存储。
- **反对权**：用户可随时停止使用本工具。

## 变更通知

如本隐私政策发生变更，我们将在本页面更新内容，并更新生效日期。
