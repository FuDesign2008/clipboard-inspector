## 服务描述

本工具是一款开源的客户端网页应用，提供以下功能：

- 通过浏览器 API 读取本地剪贴板内容
- 解析和展示剪贴板数据的类型与结构
- 将数据导出为 Markdown 或 ZIP 格式（用户主动触发）

本工具的所有功能均在用户浏览器本地运行，无需后端服务器支持。
