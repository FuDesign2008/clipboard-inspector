Privacy Policy

Effective Date: April 25, 2026

Overview

Clipboard Inspector ("this tool") is a purely client-side web tool for inspecting and viewing clipboard contents. This privacy policy explains how this tool handles user data.

Data Processing

Local Processing Only. All data processing occurs entirely within the user's browser. Reading, parsing, displaying, and exporting clipboard content does not transmit any data to external servers.

No Data Collection. This tool does not collect, store, or transmit any user data, including but not limited to:

  • Clipboard contents
  • Personal identification information
  • Usage logs
  • Device information
  • IP addresses

No Cookies. This tool does not use cookies or similar tracking technologies.

No Third-Party Data Sharing. We do not share any data with third parties because we do not collect any data.

Technical Details

This tool reads clipboard content through browser native APIs (DataTransfer / ClipboardItem). These operations occur entirely within the browser sandbox environment; data never leaves the user's device.

Hosting

This tool is hosted on GitHub Pages. GitHub Pages is an infrastructure service provided by Microsoft. For GitHub Pages data processing practices, please refer to the GitHub Privacy Statement.

User Rights (GDPR)

Although this tool does not collect any personal data, we respect user privacy rights:

  • Right to know: This policy explains how we handle data (i.e., we do not).
  • Right to access: No personal data is collected, so there is nothing to access.
  • Right to deletion: No data is stored, so no deletion is needed.
  • Right to portability: Not applicable as no data is stored.
  • Right to object: Users may stop using this tool at any time.

Changes

If this privacy policy changes, we will update this page with a new effective date.

Contact

For privacy-related questions, please contact us via GitHub Issues.
